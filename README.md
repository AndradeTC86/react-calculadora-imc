# Calculadora IMC React

## Aula de criação de uma calculadora de IMC utilizando React

Esse é um repositório criado como exercício do curso Desenvolvedor Full Stack Python da [EBAC](https://ebaconline.com.br/new/full-stack-python), desenvolvido para criar uma calculadora de IMC utilizando o React.

Uma versão do site pode ser acessada no projeto do [Vercel](https://react-calculadora-imc-snowy.vercel.app/).
